# 💌 Undangan Pernikahan Digital

Proyek ini adalah aplikasi **Undangan Digital** berbasis web yang ringan, elegan, dan responsif. Dirancang untuk memudahkan penyebaran informasi pernikahan secara modern melalui satu tautan.

## 🚀 Live Demo
Lihat undangan secara langsung di sini:
👉 ([https://undangan.vercel.app](https://undangan-fobydinda.vercel.app/))

## 🛠️ Fitur Utama
* **Desain Responsif:** Tampilan optimal di berbagai perangkat (Smartphone, Tablet, Desktop).
* **Background Music:** Memutar musik otomatis untuk menambah suasana sakral.
* **Galeri Foto:** Menampilkan momen-momen bahagia pasangan.
* **Informasi Acara:** Detail waktu dan lokasi (integrasi Google Maps).
* **Buku Tamu & Reservasi:** Memungkinkan tamu memberikan ucapan dan konfirmasi kehadiran.

## 📂 Struktur Folder
* `/foto`: Menyimpan aset gambar dan galeri pernikahan.
* `/musik`: Menyimpan file audio latar belakang.
* `index.html`: Halaman utama struktur undangan.

## 💻 Teknologi yang Digunakan
* **HTML5 & CSS3** (Custom Styling)
* **JavaScript** (Interaktivitas & Background Music)
* **Vercel** (Deployment Platform)
